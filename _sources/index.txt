sklearn-evaluation documentation
================================

scikit-learn model evaluation made easy: plots, tables and markdown reports.

Installation
************

.. code-block:: bash

   pip install sklearn-evaluation

Optional Dependencies
*********************

.. code-block:: bash

    # for generating reports
    pip install mistune
    # for tables
    pip install tabulate
